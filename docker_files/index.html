Web Hacking Sandbox

Activities

Reviews (XSS)

Try to steal an admin cookie and access the admin page.


Recipes (Deserialisation + Directory Traversal)

Try to perform a deserialisation attack to read a secret recipe. You will need to use Directory Traversal to examine the source code.



Recipe Hashing (File Inclusion)

Try to find a file inclusion vulnerability in this page. It may help to look at the source code using the Git repo (or using a vulnerability from another challenge...)



Page Viewer (SSRF)

Can you use the webpage viewer to see the contents of the local-only page?
